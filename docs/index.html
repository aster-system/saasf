SAASF

SAASF, pour Service d'Apprentissage d'Aster Système en Français, est une branche de l'organisation Aster Système. Il s'agit d'un site web, regroupant pleins de cours sur divers sujets pris par, entre autre, le créateur de l'organisation.

Réalisation

Ce site web est réalisé avec les très célèbres langages HTML et CSS, vu en SNT (même si le niveau néessaire pour créer un site de ce genre est bien supérieur à celui demandé en SNT). Pour la structuration, nous utilisation l'outil "Agatha" du logiciel "SCLS Workspace" notre organisation. La version actuelle du site web, modifiée le 12/11/2024, n'ai pas encore entièrement équipée de style, ni de structure propre. Cependant, les récents progrets de SCLS Workspace rendent tout cela possible dans très peu de temps.
