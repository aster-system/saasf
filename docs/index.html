SAASF

SAASF, pour Service d'Apprentissage d'Aster Système en Français, est une branche de l'organisation Aster Système. Il s'agit d'un site web, regroupant pleins d'informations structurées sous forme Wiki sur divers sujets pris par, entre autre, le créateur de l'organisation.

Réalisation

Ce site web est réalisé avec les très célèbres langages HTML et CSS, vu en SNT (même si le niveau néessaire pour créer un site de ce genre est bien supérieur à celui demandé en SNT). Pour la structuration, nous utilisation les outils "Agatha" du logiciel "SCLS Workspace", et "PLEOS" notre organisation. La version actuelle du site web, modifiée le 25/04/2025, n'ai pas encore entièrement équipée de structure propre. Cependant, les récents progrets de PLEOS rendent tout cela possible dans très peu de temps.
