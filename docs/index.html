SAASF

SAASF pour Service d'Apprentissage d'Aster Système en Français est une branche de l'organisation Aster Système.
Il s'agit d'un site web, regroupant pleins de cours sur divers sujets pris par le créateur d'Aster Système, Mattéo pour ceux qui me connaissent dans la vraie vie, et d'autres membres.

Pourquoi SAASF ?

Tous les cours présents sur SAASF sont rédigés par des personnes qui suivent également ces cours. Ils sont donc les plus adéquat du monde pour proposer ce service.

Réalisation

Ce site web est réalise avec les très cèlèbres langages HTML et CSS, vu en SNT (même si le niveau nécessaire pour créer un site de ce genre est bien supérieur à ce que vous avez vu).
Pour la structuration, il utilise l'outil de l'organisation mère, Aster Système, nommé SCLS Documentalist Agatha.

La version actuelle du site web, modifiée le 26/05/2024, n'ai pas encore entièrement équipée de style, ni de structure propre. Cependant, ça ne tardera pas.
