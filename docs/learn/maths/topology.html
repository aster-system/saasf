Qu'est ce qu'est la topologie ?

Les espaces topologiques

L'objet principal en topologie est l'espace topologique. En effet, un espace topologique représente un espace où les points de l'espace peuvent être définis avec les objets / propriétés basiques de topologie. Cependant, il faut définir ses objets et propriétés basiques. La plus importante est certainement l'ouvert (et le fermé). En topologie, un ouvert est une partie d'un ensemble topologique, pouvant contenir tous les points de cette partie qui ne sont pas sa frontière. Ici, la frontière d'un ouvert (dans un espace topologique) est la partie de l'espace topologique qui limite l'ouvert de l'extérieur de l'ouvert. Cette définition permet d'en introduire une autre : le voisinage. le voisinage d'un point dans un espace topologique est une partie de cette espace contenant un ouvert vers le point de départ.
